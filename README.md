# 🎓 Smart Campus Management System (Java)

## 📌 Project Overview
This is a Java-based console application designed to manage a smart campus system. It allows users to register learners, create subjects, assign subjects, and calculate total fees.

---

## 🚀 Features
- Register learners with ID, name, and email
- Create subjects with fees
- Assign subjects to learners
- Prevent duplicate enrollments
- Calculate total fee per learner
- View all learners and registrations
- Multithreading for enrollment processing
- Exception handling for invalid inputs

---

## 🛠 Technologies Used
- Java (Core Java)
- OOP Concepts
- Collections Framework (HashMap, HashSet)
- Multithreading (Runnable)
- Exception Handling

---

## 🧠 Concepts Implemented
- Encapsulation
- Custom Exception

- Data Structures Optimization
